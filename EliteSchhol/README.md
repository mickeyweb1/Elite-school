A modern and responsive school website built with React and Vite.

## Live Demo

🔗 https://elite-school-site.vercel.app/

## Features

- Responsive Design
- Home Page
- About Page
- Admission Page
- Gallery Page
- Contact Page
- Smooth Navigation
- Modern UI/UX

## Technologies Used

- React
- React Router
- JavaScript
- CSS
- Vite

## Installation

Clone the repository:

```bash
git clone https://github.com/mickeyweb1/Elite-foundation-school.git
```

Navigate to the project folder:

```bash
cd Elite-foundation-school
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
